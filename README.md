# 🍽️ NutriTrack – Application for Food Tracking

**NutriTrack** is an application designed to help users monitor their daily food intake, understand the nutritional values of the foods they consume, and receive personalized recommendations based on their health goals.  
The project aims to combine a nutritional database with an intelligent recommendation system tailored to each user's preferences, allergies, and dietary needs.

---

## 🎯 Project Goals

- Provide a simple and intuitive way to log daily meals.
- Display caloric and nutritional intake in a clear and accessible format.
- Adapt the experience based on the user’s goals (weight loss, health maintenance, muscle gain, etc.).
- Generate personalized food recommendations that consider preferences, allergies, and restrictions.
- Build a platform that can be extended with advanced analytics and artificial intelligence features.

---

## 👥 User Stories

### Regular Users
- *As a user who wants to lose weight, I want to see how many calories I’ve consumed in a day so I can adjust my diet.*
- *As a user concerned about metabolic health, I want to track my carbohydrate intake to avoid prediabetes risk.*
- *As a user with high blood pressure, I want to monitor my sodium intake to keep it within healthy limits.*
- *As a user who wants to eat healthier, I want to see how much fiber I consume daily.*
- *As an active user, I want to track my protein intake to support my fitness goals.*
- *As a user with allergies, I want the app to suggest only foods that are safe for me.*

### Nutritionists / Professionals
- *As a nutritionist, I want to recommend high-fiber or high-protein meal plans based on my clients’ needs.*
- *As a nutritionist, I want to analyze a user’s food history to offer personalized advice.*

### AI Features
- *As a user, I want the app to suggest meals based on my culinary preferences.*
- *As a user, I want to receive intelligent recommendations based on my nutritional goals.*

---

## 📦 Planned Features

- Daily food logging.
- Searchable nutritional database.
- Display of caloric and nutritional values (protein, carbs, fats, fiber, sodium, etc.).
- Personal goal setting.
- Dashboard showing user progress.
- Personalized food recommendations.
- Filtering by allergies, preferences, and restrictions.
- Generation of daily or weekly meal plans.
- Ability to save favorite foods or recipes.

---

## 🧱 Project Structure

- **User Interface** – for entering foods, viewing progress, and managing preferences.
- **Data Management System** – for storing foods, nutritional values, and user history.
- **Recommendation Engine** – for generating personalized suggestions.
- **Nutritional Analysis Module** – for calculating daily intake and comparing it with user goals.

---

## 🚀 Future Directions

- Barcode scanning for quick product identification.
- Smart notifications (e.g., “you’ve exceeded your sodium limit today”).
- A dedicated nutritionist mode with access to client data.
- Enhanced recommendations powered by more advanced AI models.
